# Gesture__app-dart

Gesture app è un'applicazione mobile sviluppata col framewrok di Flutter.Con essa potrai lanciare un dado semplicemente scuuotendo il tuo device,accendere la torcia facendo swipe ESATTAMENTE sulla icona del lampo.
Inoltre, chi ha voglia di cambiare da modalità chiara a scura quando usiamo il telefono prima di dormire stravaccati sul letto?Nessuno, infatti sarà proprio gesture app a pensare a cambiare la modalità notte appena rileva
che lo schermo del tuo device è rivolto verso il basso, in modo da permettere una visione notturna semplificata in automatico che aiuta a non rovinare la nostra vista.

Strumenti di sviluppo:
- **Flutter** - Framework per lo sviluppo mobile
- **Dart** - Linguaggio di programmazione

##  Installazione

### Prerequisiti
- Flutter SDK installato
- Un emulatore o dispositivo fisico

### Istruzioni

1. Clona il repository:
```bash
git clone https://github.com/guacho483/Gesture_app-dart.git
cd TO-DO-list
```

2. Installa le dipendenze:
```bash
flutter pub get
```

3. Esegui l'applicazione:
```bash
flutter run
```

## ISTRUZIONI D'IUSO

1. **Apri l'app** sul tuo dispositivo
2. **Segui le istruzioni** segnalate sulla schermata

## Per il test di flutter

Esegui i test dell'applicazione con:
```bash
flutter test
```
